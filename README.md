# AI Code Guide

Python コードの理解を助ける VS Code 拡張です。フローチャート表示・プロジェクト全体のマップ・コード内への AI 解説・AI編集検知ができます。

このページから拡張機能をダウンロードしてインストールします。GitHub に不慣れでも大丈夫、下の手順どおりに進めれば使えます。

---

## 手順1: ダウンロード

下のリンクをクリックすると、インストール用ファイル（`ai-code-guide.vsix`）が**自動でダウンロード**されます。常に最新版が落ちてきます。

### ▶▶▶ [ここをクリックしてダウンロード](https://github.com/tasotaku/ai-code-guide-dist/releases/latest/download/ai-code-guide.vsix) ◀◀◀

- 保存先はブラウザの「ダウンロード」フォルダです（あとで場所を聞かれるので覚えておいてください）。
- クリックしても GitHub のページが開いてしまう場合は、ページ下の方の **「Assets」** の中にある **`ai-code-guide.vsix`** をクリックしてください。

## 手順2: VS Code に入れる

1. **VS Code** を開きます。
2. キーボードで **`Cmd+Shift+P`**（Windows は **`Ctrl+Shift+P`**）を押します。画面の上に細長い入力欄が出ます。
3. そこに **`install from vsix`** と打ち、候補に出てくる **「Extensions: Install from VSIX...」** をクリックします。
4. ファイルを選ぶ画面が出るので、**手順1でダウンロードした `ai-code-guide.vsix`** を選びます。
5. しばらくすると右下に「Reload」や「再読み込み」のボタンが出ます。押すと有効になります（出ない場合は VS Code を開き直せばOK）。

これでインストール完了です。

## 手順3: 使う前の準備

拡張を動かすには Python と、AI（Claude か ChatGPT）の準備が必要です。初回だけ **[HOWTO.md](HOWTO.md)** を見ながら設定してください（コピペで進められます）。

---

## 新しいバージョンが出たら（更新）

自動では更新されません。更新したくなったら、**手順1の同じリンクからもう一度ダウンロードして、手順2をやり直す**だけです。番号が新しければ上書きされます。

各バージョンでの変更点は [CHANGELOG.md](CHANGELOG.md) にまとめてあります。

## 困ったとき

- **ダウンロードした .vsix が見つからない** → ブラウザの「ダウンロード」フォルダを探してください。
- **「Install from VSIX」が候補に出ない** → VS Code のバージョンが古い可能性があります。VS Code を最新に更新してください。
- **インストールしたのに動かない** → [HOWTO.md](HOWTO.md) の「必要なもの」を満たしているか確認してください（Python が入っているか等）。
