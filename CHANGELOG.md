# Changelog

このファイルは AI Code Guide のバージョンごとの変更点を記録します。
バージョンは [Semantic Versioning](https://semver.org/lang/ja/) に従います（新機能=minor / バグ修正=patch）。

配布は研究室内向けに GitHub Releases で行っています。更新方法は各リリースの手順に従ってください（自動更新はされないため、新しい vsix を手動で入れ直します）。

## [0.4.0] - 2026-07-04

### 追加
- **選択範囲をチャットに引用**: エディタでコードを範囲選択し、Cmd+Alt+Q（Win/Linux: Ctrl+Alt+Q）または右クリック「選択範囲をチャットに引用」で、チャットに `引用 ファイル名 L11-20` の場所参照チップが付きます。生コードを貼り付けず、どのファイルのどこかを保ったまま質問できます（コードはチップ内で展開して確認可、× で外せます）。送信するとその箇所を文脈にしてAIが回答します。

### 変更
- **サブカードの行番号表示を削除**: モジュールマップのカードを▶で展開したときのサブカードから `Lxx–yy` の行番号表示を外し、見出しと説明だけのすっきり表示にしました（クリックでの行ジャンプは従来どおり動きます）。

## [0.3.0] - 2026-07-04

### 追加
- **モジュールマップのAI説明を独立行で表示**: 関数カードの見出しは関数名のみにして、AI説明はその下の行にまとめて表示するようにした（クラスと体裁を統一）。
- **単一関数のフロー図に拡大縮小**: 「図」を開いた画面に −／＋／リセットのズーム操作を追加。大きな図も見やすくなった。
- **下線解説の表示切り替え**: 「下線: ON/OFF」で全解説をまとめて非表示／再表示できる（生成結果は保持され即座に戻せる）。各解説を個別に「隠す」／「隠した注釈も表示」で戻す操作も追加。
- **カードの範囲展開**: ツールバーの「範囲展開」からモードに入り、カードの範囲をクリックで選んでまとめてAI分解できるようにした。

### 修正
- **下線解説の位置**: 下線（symbol）の解説ボタンが下線から左にずれて表示される問題を修正し、下線の真下に来るようにした。
- **フロー図のエラー**: 「図」を開いた状態からチャットして標準表示に戻ると、フロー図が「Syntax error」になる問題を修正。
- **AI説明が出ないことがある問題**: 構造系モデルに Haiku 4.5 などを使うと、モジュールマップのカードにAI説明が生成されない問題を修正（モデル応答からの取り出しを頑健にした）。

## [0.2.2] - 2026-07-03

### 修正
- **標準タブのモジュールマップ**で、連続するトップレベルの変数定義が1つずつ別々のカードに分割されていた問題を修正。import と同じように連続する定義を1枚のカードに束ねるようにした（見出しは変数名を列挙）。

## [0.2.0] - 2026-07-03

### 追加
- **AI編集検知 → 提案式インライン解説**: AIによる一括編集を検知し、その範囲に絞ってインライン解説を提案する機能を追加。
  - 検知ロジック `bulkEditDetector`（純粋関数・単体テスト付き）
  - 動作モード設定 `off` / `suggest` / `auto`（既定は提案式）
  - ハイライト＋CodeLens による提案UI、検知範囲に絞った解説生成

### 変更
- ヘルプタブをランチャー＋クイックリファレンスに再設計、全UIから装飾絵文字を削除。
- APIキーの保存先を OS キーチェーン（SecretStorage）に移行済みの説明・削除/入れ替え手順をヘルプへ整理。

### ドキュメント
- 宣伝用デモGIF・ショーケース資料（`docs/showcase.md`）を追加。

## [0.1.2] - 2026-07-02

### 変更
- APIキー保存を SecretStorage（OSキーチェーン）に移行。設定コマンド追加・既存 settings からの自動移行に対応。
